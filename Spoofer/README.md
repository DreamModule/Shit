║   HWID Spoofer - Инструкция            ║
╚════════════════════════════════════════╝

ТРЕБОВАНИЯ:
• Windows 10/11 (x64)
• Права Администратора
• volumeid.exe (для Volume Serial) - опционально

УСТАНОВКА:
1. Распаковать в любую папку
2. (Опционально) Положить volumeid.exe рядом
3. Запустить от Администратора

ПЕРВЫЙ ЗАПУСК:
1. Выбрать [2] Тестовый режим (Dry-Run)
2. Проверить логи - какие ключи изменятся
3. Если всё ОК - выбрать [1] Полный Spoof

НАСТРОЙКА:
Редактировать config.toml:
• backup_before = true       # Создавать backup
• spoof_bios = false         # Отключить BIOS (если Secure Boot)
• spoof_volume_serial = true # Если есть volumeid.exe

ВОССТАНОВЛЕНИЕ:
1. Выбрать [3] Восстановление
2. Указать файл: registry_backup_YYYYMMDD_HHMMSS.txt

ЛОГИ:
Все операции записываются в: hwid_spoofer.log

ВНИМАНИЕ:
• Всегда делайте backup!
• Некоторые ключи защищены UEFI/TrustedInstaller
• Требуется перезагрузка для применения изменений

РЕШЕНИЕ ПРОБЛЕМ:
Если ключи не меняются:
1. Отключить Secure Boot в BIOS
2. Использовать RegOwnershipEx для захвата владения
3. Отключить проблемные ключи в config.toml
